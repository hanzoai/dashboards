# Hanzo Dashboards

Beautiful observability dashboards.

## Overview

Hanzo Dashboards provides powerful visualization and dashboarding for your AI infrastructure. Create beautiful, interactive dashboards to monitor metrics, logs, and traces from all your Hanzo observability components.

## Features

- **Rich Visualizations**: Graphs, heatmaps, tables, and more
- **Unified View**: Correlate metrics, logs, and traces
- **Alerting**: Visual alert configuration and management
- **Templating**: Dynamic, reusable dashboards
- **Annotations**: Mark events on graphs
- **Sharing**: Export and share dashboards

## Quick Start

```bash
docker run -p 3000:3000 hanzo/dashboards
```
