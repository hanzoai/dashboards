# Hanzo Dashboards

Observability and monitoring dashboards for the Hanzo platform. Based on Grafana.

## Get started

```bash
# Development
pnpm install
pnpm dev

# Build
pnpm build
```

## Documentation

See the upstream [Grafana documentation](https://grafana.com/docs/) for core functionality.
