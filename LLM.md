# LLM.md - Hanzo Dashboards

## Overview
Observability and monitoring dashboards for the Hanzo platform. Based on Grafana.

## Tech Stack
- **Language**: TypeScript/JavaScript

## Build & Run
```bash
pnpm install && pnpm build
pnpm test
```
